## Древовидный список и таблицы доступны по адресу:
```url
http://localhost/table
```
```url
http://localhost/tree
```
## Model Catalog находится:
```
/src/app/Models
```
## Запуск и остановка:
### Запуск (требуется подождать пару минут после запуска для инициализации базы с дампа)
```shell
docker-compose up -d
```

### Остановка
```shell
docker-compose stop
```

### Полное удаление
```shell
docker-compose down -v
```
